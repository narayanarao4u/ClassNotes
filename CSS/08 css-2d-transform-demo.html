🎯 CSS 2D Transform Functions Demo

🔁 translate(50px, 20px): Moves the element right and down.
🔄 rotate(45deg): Rotates the element 45 degrees.
🔍 scaleX(1.5): Scales width by 1.5x.
🔍 scaleY(1.5): Scales height by 1.5x.
🔍 scale(1.5): Scales width and height by 1.5x.
📐 skewX(30deg): Skews horizontally.
📐 skewY(30deg): Skews vertically.
📐 skew(30deg, 10deg): Skews in both directions.
🔣 matrix(1, 0.5, -0.5, 1, 30, 20): matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())
